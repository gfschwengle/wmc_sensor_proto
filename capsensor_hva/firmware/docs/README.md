# Firmware Documentation

This directory contains firmware-specific documentation.

## Contents

Place detailed technical documentation here:

- **API Documentation**: Detailed function and class references
- **Architecture**: System design and component interaction
- **Algorithms**: Detailed explanation of processing algorithms
- **Configuration**: Configuration options and parameters
- **Protocols**: Communication protocol specifications
- **Troubleshooting**: Common issues and solutions

## Documentation Format

Use Markdown (.md) files for easy viewing and version control.

## Suggested Documents

- `api.md` - API reference
- `architecture.md` - System architecture
- `algorithms.md` - Algorithm details
- `protocol.md` - Communication protocols
- `calibration.md` - Calibration procedures
- `memory-map.md` - Memory layout and usage

## Diagrams

Use tools like:
- Mermaid (embedded in Markdown)
- Draw.io
- PlantUML

## Code Documentation

In addition to these docs, use inline comments and Doxygen-style documentation in source code.
